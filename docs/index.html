Important: Running on macOS

Because Rascal is self-signed/ad-hoc signed, macOS Gatekeeper may show a quarantine warning when you first open the app. To run it:

  1. Drag Rascal to your Applications folder.
  2. Right-click (or Control-click) Rascal.app in Applications and choose Open from the menu.
  3. If macOS still warns you that the app is "damaged" or from an unidentified developer, open Terminal and run the following command to clear the quarantine flag: 
    xattr -cr /Applications/Rascal.app

or build from source

git clone https://github.com/chang-07/finder-2.git
cd finder-2
./setup-signing.sh   # once — so macOS remembers the permissions you grant
./install.sh         # builds & installs to /Applications
